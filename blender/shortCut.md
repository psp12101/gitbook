
# blender 快捷键

## 数字键盘

```
	7: top 视图
	1: front 视图
	3: side 视图
	5: perspective 视图
	0: 选择相机后切换到相机视角

```

## 添加删除

```
	shift + A:  添加物体
	x: 选择物体如果删除
	space: 搜索

```

## 平移，放缩，旋转物体或者场景

```

	鼠标滚轮按下 旋转整个场景
	+ 放大场景
	- 缩小场景
	S 放缩  用 ctrl 控制精度
	G 平移  
	G => x 沿 x 移动  
	G => y 沿 y 移动
	G => z 沿 z 移动
	G => 鼠标滚轮 动态沿着轴移动
	R 旋转

```

## modifier 扳手

```

	add modifier

	subdivision surface 细分
	wave 波浪动画 => alt + a 查看效果
	space => quick smoke  => alt + a 查看效果

```
